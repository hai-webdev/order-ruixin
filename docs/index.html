睿信资本
睿信资本总部位于马来西亚雪兰莪州，是一家以投资国际教育、移民、餐饮、艺术、软件科技等行业的跨国资本集团，现已在马来西亚、中国香港、印度尼西亚、美国科罗拉多州等地区投资 移民留学公司、华语培训学校、大学学院、美术馆、高端咖啡厅、贸易公司、非盈利基金会（社会服务）等机构数十间。
  • 创造高价值的商业市场
    通过发掘全新的价值创造方式，新的商业模式能持续不断对人们的生活和工作方式产生影响。
  • 实现多样化的商业模式
    多样化和包容型是睿信资本的发展宗旨，包容一切可能性发展开展多样化商业模式让睿信资本产生更大多的市场价值
  • 成就多个领域经营价值
    通过多样性来提升创新能力和韧性成就下一个睿信资本更好的十年。
